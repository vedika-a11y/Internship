# Story 1: SQL Query Practice

## Objective
Practice SQL queries from basic to intermediate level using a sample dataset.

## Tasks Performed
- Wrote SELECT queries using WHERE, ORDER BY and LIMIT
- Practiced different JOIN operations (INNER, LEFT, RIGHT, FULL)
- Used GROUP BY with aggregate functions (COUNT, SUM, AVG, MAX, MIN)
- Implemented HAVING clause for filtering aggregated data
- Practiced writing subqueries

## Key Concepts Learned
- Filtering data using WHERE conditions
- Combining tables using JOINs
- Aggregating data with GROUP BY
- Understanding the difference between WHERE and HAVING
- Writing efficient SQL queries

## Challenges Faced
- Understanding JOIN logic
- Writing complex GROUP BY queries
- To insert the data as the data was too large
- Dataset was in SQL format but i was using MySQL

##Solutions 
- divided the data into small parts and then inserted
- Converted all the data into mysql format

## Outcome
Developed understanding of SQL querying techniques required for data analysis
Queries to write as in business perspective
